# ZelaTech
O ZelaTech é um sistema web que moderniza a gestão de manutenção em condomínios. Moradores abrem chamados digitais com fotos e o síndico gerencia os status e o mural de avisos. Desenvolvido sob a arquitetura MVC usando Java, Servlets, JSP, JDBC, MySQL e Tailwind CSS, unindo segurança e transparência.
